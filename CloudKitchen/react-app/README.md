# CloudKitchen React App

A modern food-ordering storefront built with React, Vite, Redux, and TanStack Query.

This project is a CloudKitchen landing page experience that includes:

- chef-crafted menu browsing with category filters
- a shopping cart powered by Redux Toolkit
- toast notifications with Sonner
- responsive UI and Tailwind-based styling

## Install and run

```bash
cd CloudKitchen/react-app
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
